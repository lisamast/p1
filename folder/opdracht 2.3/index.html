onderzoek

Waarom noem je de eerste HTML-pagina; index.html?

Dat help de browser om de start pagina van een website te vinen. de webbrowser gaat altijd eerst opzoek naar een index.


Met welke HTML versie werken we nu?

HTML5, dit is de nieuwste versie van HTML.


Met welke CSS versie werken we nu?

CSS3, dit is de nieuwste versie van CSS.


Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Verschillende webbrowsers 'lezen' een website op hun eigen manier, daardoor kan een website er net wat anders uitzien in verschillende webbrowsers.


Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De meest gebruikte browser is op dit moment Google Chrome. Deze browser gebruikt 2,5 GB RAM bij normaal gebruik.
